# Judge v1.4 更新说明

**发布日期**：2025年11月10日

## 🎉 新增功能

### 模型配置管理

- ✅ 保存多个 AI 模型配置（服务类型 + 模型名称 + API Key）
- ✅ 主界面下拉菜单快速切换配置
- ✅ 配置管理界面：添加、编辑、删除配置
- ✅ 按最后使用时间自动排序
- ✅ 配置持久化存储：`~/Library/Application Support/Judge/model_configs.json`
