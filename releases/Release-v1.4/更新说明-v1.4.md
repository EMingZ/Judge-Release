# Judge v1.4 更新说明

**发布日期**：2025年11月10日

## 🎉 新增功能

### 模型配置管理

v1.4 版本引入了全新的**模型配置管理**功能，让您可以保存和快速切换多个 AI 模型配置。

#### 主要特性

- ✅ **保存多个配置**：为不同的场景创建多个模型配置
  - 例如："快速处理 - DeepSeek Chat"、"深度分析 - ZhipuAI Thinking"
  
- ✅ **一键切换**：在主界面通过下拉菜单快速切换配置
  - 无需重复输入服务类型、模型名称和 API Key
  
- ✅ **配置管理界面**：专门的管理窗口
  - 添加、编辑、删除配置
  - 配置名称唯一性验证
  - 悬停显示操作按钮（编辑/删除）
  
- ✅ **智能排序**：按最后使用时间排序
  - 常用配置自动置顶
  
- ✅ **完整信息**：每个配置包含
  - 自定义名称
  - 服务类型（DeepSeek / ZhipuAI）
  - 模型名称
  - API Key（安全存储）
  - ZhipuAI 思考模式开关
  - 创建时间和最后使用时间

#### 使用方法

1. **创建配置**
   - 点击主界面的"管理模型"按钮
   - 点击"添加配置"
   - 填写配置信息并保存

2. **切换配置**
   - 在"模型配置"下拉菜单中选择
   - 所有参数自动加载

3. **编辑配置**
   - 在管理界面悬停在配置上
   - 点击编辑按钮修改

4. **删除配置**
   - 在管理界面悬停在配置上
   - 点击删除按钮（带确认对话框）

## 🔧 技术改进

### 文件结构
新增配置文件：
```
~/Library/Application Support/Judge/model_configs.json
```

保存内容：
- 所有模型配置
- 当前选中的配置 ID
- JSON 格式，可读性强

### UI 优化
- 主界面移除冗余的服务/模型/API Key 输入区域
- 使用简洁的下拉选择器 + 管理按钮
- 配置管理界面采用现代化设计
  - 空状态提示
  - 选中状态高亮
  - 悬停交互

## 🆙 升级指南

### 从 v1.3 或更早版本升级

1. 下载 `Judge-v1.4-Installer.dmg`
2. 替换 Applications 文件夹中的 Judge.app
3. 首次启动后，点击"管理模型"创建新的配置
4. 输入您的 API Key 和模型设置

**注意**：v1.4 采用全新的配置管理系统，旧版本的配置不会自动迁移。请手动创建新配置。
